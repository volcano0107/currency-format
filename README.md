# currency-format
=========

A small library that format currency 
货币格式化

## Installation

  `npm install @wangcancan/currency-format`

## Usage

    var currencyFormatter = require('@wangcancan/currency-format');

    var formattedCurrency = currencyFormatter(35666);
  
  
  Output should be `$35,666`


## Tests

  `npm test`
